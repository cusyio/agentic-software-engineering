Agentic software development is transforming the way Python code is written.
However, to ensure that the agents use a modern toolchain comprising `uv
<https://python-basics-tutorial.readthedocs.io/de/latest/libs/install.html#uv>`_,
`Ruf <https://www.python4data.science/de/latest/productive/qa/ruff.html>`_, `ty
<https://docs.astral.sh/ty/>`_, `prek <https://prek.j178.dev/>`_ and `pytest
<https://python-basics-tutorial.readthedocs.io/de/latest/test/pytest/index.html>`_,
they must be configured accordingly. This tutorial covers the configuration,
project initialisation and monitoring measures that ensure AI-generated code
remains maintainable.

Pull Requests
-------------

If you have any suggestions for improvements or additions, I recommend that you
create a `fork <https://github.com/cusyio/agentic-software-engineering/fork>`_
of our `GitHub repository
<https://github.com/cusyio/agentic-software-engineering/>`_ and make your
changes there. You are also welcome to submit a pull request. Provided the
changes included are small and atomic, we’ll be happy to review your
suggestions.
